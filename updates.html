🚀 A New Version is Coming!

The UmbraGreet bot is evolving! We are working on a major update that will enhance user introductions and improve community engagement.

🛠 What's New?

  • ✨ **Improved User Profiles** – Users can add more details to their introduction, such as favorite games, skills, and achievements.
  • 🖼️ **Profile Images & Custom Avatars** – Option to display a custom profile image alongside user introductions.
  • 📌 **Pinned Introductions** – Server admins can pin important introductions for easy reference.
  • 📊 **User Interaction Stats** – Track how many times a user has introduced themselves or updated their profile.
  • 📝 **Custom Introduction Templates** – Server admins can set predefined introduction templates for a structured format.
  • 🔄 **Auto-Update Profiles** – Users can refresh their profile details easily without needing to re-enter all information.

📅 Expected Release Date

The new version is in development and will be available **soon**. Stay tuned for the official release announcement!

⚠️ Important Notice: Some features may change before the final release. The update may require a **database reset**, so please **backup important data** if needed.
