✨ Key Features

🌟 Profile Management

Users can introduce themselves using `/intro`, saving their details like name, hobbies, and more.

📜 User Info Retrieval

Retrieve any user’s profile using `/get_info` to see their introduction details.

🔒 Command Restrictions

Admins can allow/disallow commands in specific channels using `/allow` and `/disallow`.

⚡ Database Storage

All user data is securely stored in a local SQLite database, ensuring persistent user information.

📚 How It Works

UmbraGreet enhances Discord community management by automating user introductions, command restrictions, and more.

Admins can use `/allow` and `/disallow` to control which channels can use certain commands.
