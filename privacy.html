Privacy Policy

How UmbraGreet handles and protects user data.

1. Introduction

UmbraGreet is committed to protecting your privacy. This Privacy Policy explains what data we collect, how we use it, and how we protect it.

2. Data Collection

UmbraGreet does not collect or store any personal data beyond what is necessary for bot functionality. User profile information, such as names and hobbies, is stored **only** within the server’s database.

3. Message & Command Processing

All bot commands and messages processed by UmbraGreet occur in real-time and are **not** stored. The bot does not log private messages or retain conversation history.

4. Data Security

All stored data is protected within the Discord server’s database and is not shared externally. However, Softvionix is **not responsible** for data loss due to server resets or updates.

⚠️ Important Notice: Some bot updates may reset the database, erasing stored user profiles. Softvionix **is not responsible** for lost data. Regular backups are recommended.

5. Third-Party Services

UmbraGreet does not share or sell user data to third parties. The bot operates entirely within Discord’s API and follows all Discord security guidelines.

6. Changes to Privacy Policy

Softvionix reserves the right to update this Privacy Policy as needed. Changes will be posted on this page, and continued use of the bot implies acceptance of any modifications.
