2. General Usage

UmbraGreet is a Discord bot designed to enhance community engagement and moderation by providing a structured user introduction system and command restriction features. The bot allows users to introduce themselves using /intro, storing details like name, age, hobbies, and traits for easy retrieval via /get_info. Administrators can manage bot permissions using /allow and /disallow, ensuring that commands work only in designated channels. This functionality is particularly useful for servers that wish to maintain organized discussions and controlled command usage. The bot also automates moderation tasks, ensuring compliance with server rules while allowing admins to customize which commands are available. All user profile data is stored securely in a local SQLite database, but data loss may occur during major bot updates, as the database may reset. The bot is regularly updated to improve performance, add new features, and enhance security, ensuring a smooth and efficient experience for Discord communities.

3. Bot Commands & Permissions

  • /intro: Allows users to introduce themselves.
  • /get_info: Retrieves user profile information.
  • /allow: Enables commands in specific channels.
  • /disallow: Disables commands in specific channels.

Only administrators can modify command restrictions.

4. Data Collection & Privacy

UmbraGreet values user privacy and is designed to operate with minimal data collection. The bot only stores essential user profile information provided through the /intro command, including name, nickname, age, gender, country, hobbies, traits, and additional details, which are stored within the Discord server’s database for easy retrieval via /get_info. No personally identifiable information (PII) is collected, and the bot does not track, analyze, or share user data outside of the server. The data is stored locally in an SQLite database, ensuring that it remains confined to the server where the bot is used. Additionally, UmbraGreet does not monitor private messages, voice channels, or any other Discord activities beyond the interactions initiated through its commands. While the bot aims to maintain data integrity, periodic updates or major bot revisions may require database resets, which could lead to loss of stored profiles, and Softvionix holds no responsibility for lost data due to such updates. Users should be aware that Discord server administrators have control over bot data and may delete or reset stored user information at their discretion. By using UmbraGreet, users acknowledge and agree to these data practices, ensuring transparency and trust in how their information is handled.
